# Java Swing シンプルカウンターアプリ

Java の学習の一環として作成した、
0〜9999 までカウントできるシンプルな GUI アプリです。

## 🔧 機能

「カウント」ボタンを押すと 1 ずつ増える

上限は 9999

「リセット」ボタンで 0 に戻す（確認ダイアログつき）

## 👨‍💻 開発環境

Java 21
Eclipse
Swing

## 💡 目的

プログラミング学習（Java／Swing）の成果物として作成。
ChatGPT からアドバイスを受けながら実装しました。

※ EXE 化などの配布用ビルドは行っていません。

## 📁 実行方法

Eclipse またはコマンドラインで CounterApp.java を実行するだけで動きます。
